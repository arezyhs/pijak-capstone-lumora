# Machine Learning Workspace

Folder ini disiapkan untuk eksperimen model adaptive learning.

## Tahap 1

- `src/recommender.py` berisi baseline rule-based recommendation.
- `data/` untuk dataset lokal yang tidak perlu masuk git jika berukuran besar.
- `notebooks/` untuk EDA dan eksperimen.
- `models/` untuk artifact `.joblib` atau `.pkl`.

## Kandidat Dataset

- Student Performance Dataset.
- xAPI-Edu-Data.
- Kaggle Education Dataset.

## Target Model

- Klasifikasi performa siswa.
- Rekomendasi topik remedial atau enrichment.
- Adaptive difficulty untuk quiz berikutnya.
