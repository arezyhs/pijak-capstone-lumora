# Milestones

## Minggu 1 - Inisialisasi dan Perencanaan

- Setup repository dan struktur monorepo.
- Dokumentasi arsitektur sistem.
- API contract awal.
- Wireframe/dashboard awal.
- Eksplorasi kandidat dataset.

## Minggu 2 - Pengembangan Awal

- EDA dan preprocessing data.
- Baseline model.
- Backend dummy API dan database awal.
- Frontend statis dengan mock data.

## Minggu 3 - Integrasi Sistem

- Integrasi model ke FastAPI.
- Endpoint rekomendasi aktif.
- Frontend mengambil data dari backend.

## Minggu 4 - Pengujian dan Penyempurnaan

- End-to-end testing.
- Adaptive quiz.
- Optimasi UI/UX dan error handling.

## Minggu 5 - Finalisasi

- Deployment MVP.
- Dokumentasi teknis dan laporan akhir.
- UAT dan persiapan demo.
