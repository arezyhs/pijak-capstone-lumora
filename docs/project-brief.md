# Project Brief

## Nama Produk

Lumora - Adaptive AI Learning Path.

Nama Lumora diambil dari "Lumen" yang berarti cahaya dan "Mora" yang merepresentasikan belajar. Nama ini membawa gagasan produk sebagai cahaya penuntun untuk proses belajar yang lebih personal.

## Masalah

Pembelajaran konvensional sering memakai pendekatan yang sama untuk semua siswa, padahal kemampuan, kecepatan belajar, dan kelemahan tiap siswa berbeda.

## Solusi

Platform pembelajaran adaptif berbasis AI yang menganalisis performa belajar siswa dan memberi rekomendasi materi serta latihan sesuai kebutuhan personal.

## MVP

- Dashboard siswa untuk progress, rekomendasi materi, hasil quiz, dan personal learning path.
- Dashboard guru/admin untuk monitoring performa siswa.
- REST API untuk hasil evaluasi, progress tracking, rekomendasi materi, dan adaptive quiz.
- Model ML awal untuk analisis performa dan rekomendasi tingkat kesulitan.

## Pertanyaan Riset

- Faktor apa yang paling mempengaruhi performa belajar siswa?
- Bagaimana AI membantu personalisasi pembelajaran?
- Seberapa efektif adaptive learning dibanding metode pembelajaran umum?
