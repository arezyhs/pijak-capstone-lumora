# Lumora Frontend

Frontend React TypeScript untuk Lumora, platform adaptive learning berbasis AI.

Nama Lumora diambil dari "Lumen" yang berarti cahaya dan "Mora" yang merepresentasikan belajar.

## Development

```bash
npm install
npm run dev
```

URL lokal: `http://localhost:5173`.

## Build

```bash
npm run build
```
